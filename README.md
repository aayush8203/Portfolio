# Ayush Mishra | Full Stack Developer Portfolio 🚀

A futuristic, modern, and highly interactive personal portfolio website built to showcase my projects, skills, and experience as a Full Stack Web Developer.

![Portfolio Preview](./preview.png)

## ✨ Features
- **Futuristic & Dark Theme**: Premium aesthetic with deep space backgrounds, glowing orbs, and glassmorphism.
- **Micro-Animations**: Custom scroll reveals, typing animations, particle cursor trails, and 3D hover effects.
- **Component Sections**:
  - Hero Section (Typing effect, Resume Download)
  - About Me (Animated floating Avatar)
  - Tech Stack (Interactive categorized tabs with animated progress bars)
  - Projects (Premium project cards with glow effects)
  - Services (3D tilt cards detailing offerings)
  - Experience (Timeline format with animated dots)
  - Contact (Fully styled form and multi-channel links)
- **Responsive Layout**: Bootstrap 5 grid system scaling beautifully across all devices (Mobile-First approach).
- **SEO Optimized**: Meta tags, semantic HTML5, and high performance tuning included.

## 🛠️ Tech Stack Built With
- **Frontend Core**: HTML5, Vanilla JavaScript (ES6+), Modern CSS3 (`var`, `flex`, `grid`, animations).
- **CSS Framework**: Bootstrap 5 (Spacing utilities and Grid System)
- **Icons**: Bootstrap Icons
- **Fonts**: Google Fonts (`Outfit` for headings/ui, `JetBrains Mono` for developer aesthetic bits).

## 🚀 Quick Start & Deployment

### Run Locally
Simply open the `index.html` file in any modern web browser to view the project:
```bash
# Optional: Use a local server to avoid CORS issues on some local JS features
npx -y serve .
```

### Vercel Deployment (Recommended)
This repository is pre-configured for Vercel. 
1. Create a GitHub repository and push this code.
2. Sign in to [Vercel](https://vercel.com/) and click **New Project**.
3. Import your GitHub repository.
4. Click **Deploy**. Vercel will automatically configure and serve the site with optimal caching using the provided `vercel.json`.

## ⚙️ Configuration & Personalization

Before deploying, make sure to update the following:
1. `index.html`: Open and replace standard placeholders (e.g. `github.com/ayushmishra`, `ayushmishra@example.com`, WhatsApp Number `91XXXXXXXXXX`).
2. Add your own `resume.pdf` into the root directory of the project. This is linked natively to the "Download Resume" buttons.
3. Update specific project thumbnails (or swap gradients) in `css/style.css` if deploying new projects.
